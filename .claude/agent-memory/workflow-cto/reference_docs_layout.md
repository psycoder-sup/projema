---
name: Docs layout for project-managment repo
description: Where PRDs and SPECs live in this repo, so future runs save outputs to the right path.
type: reference
---

PRDs and SPECs live at `docs/feature/<feature-slug>/<feature-slug>-prd.md` and `docs/feature/<feature-slug>/<feature-slug>-spec.md`.

Example: `docs/feature/sprint-todo-management/sprint-todo-management-prd.md` and `docs/feature/sprint-todo-management/sprint-todo-management-spec.md`.

**How to apply:** When asked to produce a SPEC for a PRD, place the SPEC alongside the PRD in the same `docs/feature/<feature-slug>/` directory, using the `-spec.md` suffix.
